# Data Structures and Algorithms

This repository contains my implementations and progress through the study of efficient algorithms and data structures, focusing on performance and real-world applications.

## Projects

### [Union-Find / Percolation](./Union-Find/Percolation)
An implementation of the **Weighted Quick Union with Path Compression** algorithm to solve the scientific problem of percolation.
- **Key Features**: Path compression, tree weighting, and Monte Carlo simulation.
- **Complexity**: Near-constant time per operation ($O(\alpha(N))$).
